# Animated Challenges App

This React application is built to explore advanced UI animations using Framer Motion. The goal was to create a smooth, motion-driven user experience with polished transitions, scroll effects, and interactive micro-animations.

## üé¨ Animation Highlights (Framer Motion)

### 1. Layout Animations
- Challenge cards smoothly reposition when switching tabs  
- Items animate naturally when added or removed  
- Details expand and collapse with smooth transitions  

These animations prevent abrupt layout shifts and create a fluid experience.

---

### 2. Shared Layout Animation
- The active tab indicator slides smoothly between tabs  
- Improves visual continuity and navigation clarity  

---

### 3. Enter & Exit Animations
- Challenge list transitions  
- Challenge item removal  
- Modal open and close  
- Empty state animations  

Elements animate out before unmounting for a polished experience.

---

### 4. Staggered Animations
- Challenge items animate in sequentially  
- Creates a cascading and professional UI effect  

---

### 5. Scroll-Based Parallax
- Parallax background movement  
- Hero text scaling and shifting  
- Scroll-based opacity transitions  

Creates a dynamic and immersive landing section.

---

### 6. Floating Idle Animation
- Subtle looping animation in the hero section  
- Keeps the interface visually alive  

---

### 7. Micro-Interactions
- Button hover scaling  
- Tap feedback  
- Card hover lift  
- Badge pulse animation  

These small details improve responsiveness and polish.

---

### 8. Modal Animation
- Backdrop fade-in and fade-out  
- Slide-up entrance animation  
- Blur-to-focus transition  

Provides a modern dialog experience.

---

### 9. Form Validation Feedback
- Shake animation on invalid form submission  
- Clear visual feedback for user errors
